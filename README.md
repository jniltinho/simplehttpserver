# SimpleHTTPServer

Go alternative of python SimpleHTTPServer with Fiber

## Usage

```bash
Usage of ./simplehttpserver:
  -listen string
        Address:Port (default "0.0.0.0:8000")
  -path string
        Folder (default "current path")
```

## Build

```bash
git clone https://github.com/jniltinho/simplehttpserver.git
cd simplehttpserver
go mod tidy
make
```
